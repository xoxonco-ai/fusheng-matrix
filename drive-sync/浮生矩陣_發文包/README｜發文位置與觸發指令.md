# 浮生矩陣｜在哪邊發文 & 觸發指令總表

## 一、安裝（電腦上做一次就好）

1. 雙擊「安裝skills_雙擊我.command」（被 macOS 擋下就：右鍵 → 打開）
2. Chrome 安裝「Claude in Chrome」擴充功能，登入 FB / IG / Threads
3. 打開 Claude Code，切到放「發文包」的資料夾

## 二、每天的發文位置（照時間順序）

- 08:00｜1_Threads短句 → Threads：複製貼上，或用指令自動發
- 12:30｜2_IG輪播貼文 → Instagram：先產 7 張圖 → IG 選多張圖＋貼 caption
- 15:00｜3_Threads延伸 → Threads：複製貼上，或自動發
- 17:00｜4_IG限動互動 → IG 限時動態：手機 IG App 加投票/問答貼紙（最快）
- 20:00｜5_IG-Reels → IG Reels：先做影片 → 上傳＋貼【Reels Caption】
- 20:00 或 21:30｜6_Facebook長文 → FB 粉專：複製貼上，或自動發
- 21:00｜7_vocus長文 → vocus.cc：手動貼（【標題】行＝文章標題；vocus 無 API）

日期對照：D01 = 7/6（週一）～ D14 = 7/19（週日）

## 三、Claude Code 觸發指令（複製著用）

- **自動發文（social-post）**：「用 social-post 讀 D01 的 1_0800_Threads短句.md，發到 Threads」→ 它顯示草稿 → 妳回「確認」才會真的發出去
- **產輪播圖（social-cards）**：「用 social-cards 讀 D01 的 2_1230_IG輪播貼文.md，照【輪播說明】做成 7 張 1080x1350 輪播圖」
- **做 Reels 影片（seedance / ai-media-generator）**：「用 seedance-20 照 D01 的【影片腳本】寫 15 秒直式（9:16）影片提示詞」／「用 ai-media-generator 把這段腳本做成影片，療癒系、柔光、慢節奏」
- **發文前審稿（x-mentor）**：「用 x-mentor 幫我審這條 Hook，打分＋給 3 個改寫版」
- **懶人一句話（全流程）**：「今天是 D03。讀發文包 D03 資料夾，照時間把 1、3、6 發出去（每篇發前給我確認），2 先產圖給我看」

## 四、注意事項

- vocus 和 IG 限動最適合手動：vocus 無 API、限動貼紙要 App 操作
- social-post 靠 Chrome 模擬人工操作：一天別連發太多、平台改版時可能偶爾失靈
- 人在外面時：打開這個 Drive 資料夾的「W1 全文」「W2 全文」，找到當天段落複製，直接在手機 App 貼文
