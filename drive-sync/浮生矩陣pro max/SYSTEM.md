## 三、素材卡格式

```
【素材卡 2026-0708-01】
類型：故事片段／金句／課程點子／學員回饋
摘要：20 字以內白話
原文：一字不改原封照貼
可用房間：見幻筆記／經在人間／浮生解碼／登出指南／回家（最多選二）
```

規則：原文不改寫；改寫發生在共創層。「可用房間」直接沿用 vocus 五房間分類，不另創分類系統。

## 四、一天與一週的循環（貼合 90 天排程）

- 每天：照 README 入口——發文用 03_啟動Prompt/短版_每日內容版.md；產出對照 90 天排程表當日列
